Muhammad Izhar Ahmed - Science & STEM Educator Portfolio

How to launch:
1. Open index.html locally to preview.
2. To publish on Netlify, drag this whole folder into Netlify Deploy manually.
3. To publish on GitHub Pages, upload all files to a repository and enable Pages.

Important files:
- index.html: website structure
- style.css: website styling
- script.js: smooth effects, gallery filters, image lightbox
- assets/images: portfolio images
- assets/docs/Muhammad_Izhar_Ahmed_CV.pdf: downloadable CV
